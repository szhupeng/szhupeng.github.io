Java权限控制实现

发表于 2018-07-29 | 分类于 Java | 阅读次数:

本篇是实现多权限控制的一种思路，基于按位与运算。

原理

用户的权限分很多种，比如添加，删除，修改等。可以为每一个特定权限设置一个唯一的整数值，如果用户拥有权限值K，H，D，R对应的权限，则用户最终对应的权限值为：

UP = 2K + 2H + 2D + 2R

或

UP = 1<<K + 1<<H + 1<<D + 1<<R

通过2的权限和来代表权限值，当要验证用户是否有某个权限的时候，只需要将权限和值与该权限对应的整数值进行按位与，如果结果还是为该整数值，则代表用户拥有该权限，否则就无权限。这种算法的一个优点是速度快。可以同时处理N个权限，设置N种角色。

示例

假如，为每一个操作设定一个唯一的整数值，比如：
　　删除A—0
　　修改A—1
　　添加A—2
　　删除B—3
　　修改B—4
　　添加B—5
　　…
理论上可以有多个操作，这取决于用于储存用户权限值的数据类型。这样，如果用户A有权限：添加A—2；删除B—3；修改B—4。那用户的权限值UP=22+23+24=28，也就是2的权的和。化成二进制可以表示为11100。如果要验证用户是否有删除B的权限，就可以通过按位与运算来实现。
        11100
    & 01000
    ————————
        01000 == 8(十进制)　!=　23


当用户有操作权限时，运算出来的结果就会等于这个操作需要的权限值。
同理，如果要验证是否有删除A—0的权限，可以用：UP & ((int)Math.pow(2, 0))，即：
        11100
    & 00001
    ————————
        00000 == 0(十进制)　!=　20

如果想验证是否同时有删除A—0和删除B—3的权限，可以用UP&(20+23)==(20+23)?true:false。
